# About
---

legallead.jdbc.connector is an micro-orm. 
It connects to backen Postgre SQL storage using Coachroach for serverless hosting.

## How to Use
---

This package is build targeting NETCORE projects.  
Currently built using NET6, with a potential upgrade to NET8 in the near future.
